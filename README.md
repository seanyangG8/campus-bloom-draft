# Campus Bloom

Admin and analytics dashboard built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting started

Prerequisites: Node.js and npm installed.

```sh
git clone https://github.com/seanyangG8/campus-bloom-draft.git
cd campus-bloom-draft
npm install
npm run dev
```

## Available scripts

- `npm run dev` – start the Vite dev server.
- `npm run build` – create a production build.
- `npm run preview` – preview the production build locally.
- `npm run lint` – run eslint.
- `npm run test` / `npm run test:watch` – run vitest suites.

## Deployment notes

Build the app with `npm run build`; deploy the contents of `dist/` to your hosting provider (e.g., Netlify, Vercel, or any static host).
